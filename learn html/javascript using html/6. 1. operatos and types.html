        operators examples results
        && logical(AND) examples : x && y the && operator is used for logical AND operations. It returns true if both operands are true, and false otherwise.
        || logical(OR) examples : x || y the || operator is used for logical OR operations. It returns true if at least one of the operands is true, and it returns false if both operands are false
